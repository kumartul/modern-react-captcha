modern-react-captcha Documentation

Installation

Using npm:

npm install modern-react-captcha

Using yarn:

yarn add modern-react-captcha

Import ReactCaptcha Component

import ReactCaptcha from 'modern-react-captcha';

ReactCaptcha with reload functionality

import React from 'react';

import ReactCaptcha from 'modern-react-captcha';

import reloadIcon from './images/reloadIcon.svg';

const App = () => {
	const handleSuccess = () => alert('Captcha matched!');
	const handleFailure = () => alert('Captcha does not match');

	return (
		<div>
			<ReactCaptcha
				charset='ulns'
				length={6}
				color='white'
				bgColor='black'
				reload={true}
				reloadText='Reload Captcha'
				reloadIcon={reloadIcon}
				handleSuccess={handleSuccess}
				handleFailure={handleFailure} />
		</div>
	);
}

export default App;

Props

charset

Specifies which character set(s) to use to generate the captcha

We have defined the first letter of some standard character sets to use while generating the captcha:

Character Set Code
Uppercase u/U
Lowercase l/L
Number n/N
Special Character s/S

If the charset prop contains any of these letters, then it will generate the captcha by mixing the characters from the specified character sets.

Example

import React from 'react';

import ReactCaptcha from 'modern-react-captcha';

const App = () => {
	const handleSuccess = () => alert('Captcha matched!');
	const handleFailure = () => alert('Captcha does not match');

	return (
		<div>
			{/* This will generate a captcha using uppercase letters and lowercase letters only */}
			{/* Default value of charset is 'ulns' */}
			<ReactCaptcha
				charset='ul'
				length={6}
				color='white'
				bgColor='black'
				handleSuccess={handleSuccess}
				handleFailure={handleFailure} />
		</div>
	);
}

export default App;

If you want to use all the character sets, either don't pass the charset prop or pass 'ulns' (order doesn't matter).

length

Specifies the length of the captcha

Example

import React from 'react';

import ReactCaptcha from 'modern-react-captcha';

const App = () => {
	const handleSuccess = () => alert('Captcha matched!');
	const handleFailure = () => alert('Captcha does not match');

	return (
		<div>
			{/* Since, no charset is passed here, therefore, all the character sets will be used to generate the captcha */}
			{/* This will generate a captcha 5 characters long */}
			{/* Default value of length is 6 */}
			<ReactCaptcha
				length={5}
				color='white'
				bgColor='black'
				handleSuccess={handleSuccess}
				handleFailure={handleFailure} />
		</div>
	);
}

export default App;

length must be a positive integer else a TypeError will be thrown

length must be greater than 0 else NegativeLengthError will be thrown

length must be less than 256 else MaxLengthExceededError will be thrown

color

Specifies the color of the captcha

Example

import React from 'react';

import ReactCaptcha from 'modern-react-captcha';

const App = () => {
	const handleSuccess = () => alert('Captcha matched!');
	const handleFailure = () => alert('Captcha does not match');

	return (
		<div>
			{/* Since, no length is passed here, therefore, captcha will be 6 characters long}
			{/* The color of the characters of the captcha will be white */}
			{/* Default value of color is 'black' */}
			<ReactCaptcha
				charset='uln'
				color='white'
				bgColor='black'
				handleSuccess={handleSuccess}
				handleFailure={handleFailure} />
		</div>
	);
}

export default App;

If you set the value of the color prop to 'random', then the color of each character of the captcha will be generated randomly

bgColor

Specifies the background color of the captcha

Example

import React from 'react';

import ReactCaptcha from 'modern-react-captcha';

const App = () => {
	const handleSuccess = () => alert('Captcha matched!');
	const handleFailure = () => alert('Captcha does not match');

	return (
		<div>
			{/* Since, the value of the color prop is set to 'random', therefore, the color of each character of the captcha will be generated randomly */}
			{/* The background color of the captcha will be yellow */}
			{/* Default value of bgColor is 'white' */}
			<ReactCaptcha
				charset='uln'
				color='random'
				bgColor='yellow'
				handleSuccess={handleSuccess}
				handleFailure={handleFailure} />
		</div>
	);
}

export default App;

If you set the value of the bgColor prop to 'random', then the background color of the captcha will be generated randomly

reload

Specifies whether the captcha can be reloaded or not

Example

import React from 'react';

import ReactCaptcha from 'modern-react-captcha';

const App = () => {
	const handleSuccess = () => alert('Captcha matched!');
	const handleFailure = () => alert('Captcha does not match');

	return (
		<div>
			{/* Since, the value of the bgColor prop is set to 'random', therefore, the background color of the captcha will be generated randomly */}
			{/* Since, the reload prop is passed here, the users will be able to reload the captcha */}
			{/* Default value of reload is false */}
			<ReactCaptcha
				charset='uln'
				color='random'
				bgColor='yellow'
				reload={true}
				handleSuccess={handleSuccess}
				handleFailure={handleFailure} />
		</div>
	);
}

export default App;

reloadText

Specifies the text to be shown to reload the captcha

Example

import React from 'react';

import ReactCaptcha from 'modern-react-captcha';

const App = () => {
	const handleSuccess = () => alert('Captcha matched!');
	const handleFailure = () => alert('Captcha does not match');

	return (
		<div>
			{/* Since, reload prop is passed here, therefore, the users will be able to reload the captcha */}

				charset='uln'

				bgColor='yellow'
				reload
				reloadText='Reload'
				handleSuccess={handleSuccess}
				handleFailure={handleFailure} />
		</div>
	);
}

export default App;

handleSuccess

Specifies the function to call if the captcha matches

This function is invoked every time the value in input field changes

Example

import React from 'react';

import ReactCaptcha from 'modern-react-captcha';

const App = () => {
	const handleSuccess = () => alert('Captcha matched!');
	const handleFailure = () => alert('Captcha does not match');

	return (
		<div>
			{/* If the captcha matches, the handleSuccess function will be called and as a result, an alert with the text 'Captcha matched!' will be shown */}
			{/* There is no default value of handleSuccess */}
			{/* You must pass a function as the value of handleSuccess */}
			<ReactCaptcha
				charset='ulns'
				color='#0082e6'
				bgColor='whitesmoke'
				reload={true}
				reloadText='Reload'
				handleSuccess={handleSuccess}
				handleFailure={handleFailure} />
		</div>
	);
}

export default App;

This function is invoked every time the value in input field changes
