1. Clone the repository

```bash
git clone https://github.com/kumartul/r-captcha.git
```

2. Change the working directory

```bash
cd r-captcha
```

3. Install dependencies

```bash
npm install
```

4. Switch to example directory to see the example

```bash
cd example
```

5. Run the example

```bash
npm run start
```
